# Responsive Portfolio Landing Page

A modern, mobile-first portfolio website built as a front-end development assignment.  
It showcases clean design, accessibility, responsive layout techniques, dark mode support, and smooth interactions.

**Live Demo:** (https://github.com/TheFuhrer-leader/advanced-css-page)

## Features Implemented

- Mobile-first responsive design with at least two breakpoints
- CSS **Grid** used for the major projects section layout
- CSS **Flexbox** used for the navbar and hero content alignment
- Fully responsive navigation:
  - Hamburger menu with slide-in panel on mobile
  - Horizontal links on desktop
- Dark mode toggle button with CSS custom properties (variables)
- Theme persistence using `localStorage`
- At least two hover transitions (buttons and project cards – lift + shadow + color change)
- One simple entrance animation (fade-in-up on hero section)
- Accessibility best practices:
  - Single `<h1>` on the page
  - Visible keyboard focus styles
  - Sufficient color contrast in both light and dark modes
- Professional typography using Google Fonts (Inter)
- Semantic HTML structure
